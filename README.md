# Amazed

Amazed is a small C project that computes and simulates the fastest way for robots to travel from a start point to a destination through a maze-like map. The program focuses on parsing map input, finding efficient routes, and stepping robots through those routes to reach the target as quickly as possible.

## Features

- Parser for map input and validation
- Pathfinding algorithm(s) to compute efficient routes
- Step-by-step simulation of robot movement
- Reusable utility libraries in `lib/` (linked list, helpers)
- Unit tests under `tests/` and `unit_tests/`
- CICD build tests inside .github/

## Build

Build the project from the repository root:

```bash
make
```

This produces the project binary (`amazed`).

## Usage

Run the program with a map/input file:

```bash
cat <map_file> | ./amazed
```

Input maps must follow the project's expected format.

## Project layout

- `src/` — Core sources: parser, initialization, main, linking/path logic
- `include/` — Public headers and shared structs
- `lib/` — Small reusable libraries (linked-list, helpers)
- `tests/`, `unit_tests/` — Test code and coverage artifacts

## Tests

Check the `Makefile` for available test targets. Unit tests are provided in `tests/` and `unit_tests/`.

## Contributing
